# Data

This directory contains all data files used by the CAB course search engine.

## Files

| File | Description |
|------|-------------|
| `openai.key` | OpenAI API key (plain text, single line) |
| `courses_overview.json` | Array of semesters, each with a `results` array of course sections (~131k sections, Summer 2016 -- Spring 2026) |
| `embeddings.jsonl` | One JSON line per section: `{"k":"crn:srcdb","v":[...256 floats...]}`. Generated by `scripts/embed_courses.js` using `text-embedding-3-small` |
| `index.json` | Collapsed course index keyed by course code, with embeddings and section metadata. Generated by `scripts/indexer.js` |
| `idmap.json` | Array mapping FAISS vector indices to course codes. Generated by `scripts/indexer.js` |
| `faiss_readable.json` | Human-readable FAISS data (code, title, vector per course). Generated by `scripts/indexer.js` |
